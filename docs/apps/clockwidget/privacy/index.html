简约日历隐私政策

最后更新：2026-04-08。本文说明当前版本简约日历会访问哪些系统能力，以及这些数据如何被使用。

适用范围

本政策适用于简约日历 iOS App 及其相关桌面小组件。

我们会访问哪些系统能力

位置 日历 提醒事项 通知
  • 位置：用于获取当前位置并查询天气；仅在你使用相关功能时请求。
  • 日历：用于读取和展示系统日程，以及通过系统界面创建新日程。
  • 提醒事项：用于读取、展示和管理系统提醒事项。
  • 通知：用于计时器、提醒相关本地通知与提示。

数据如何处理

主要在设备本地处理的数据

  • 日历事件
  • 提醒事项
  • 计时器模板、运行状态和历史记录
  • 小组件共享缓存
  • 外观和模块设置

当前版本没有自建账号系统，也没有把上述内容上传到开发者自建服务器。

可能发送到第三方或平台服务的数据

天气功能可能会将查询所需的位置数据发送给天气数据服务，以返回天气结果。当前默认主数据源为 Apple Weather，失败时可能自动回退到 Open-Meteo。

我们不会做什么

  • 不会出售你的个人信息
  • 不会基于你的数据做广告跟踪
  • 不会在未说明的情况下把日历、提醒事项内容上传到开发者自建服务器

当前未接入的能力

  • 广告 SDK
  • 行为埋点 / 统计分析 SDK
  • 第三方登录
  • 跨设备账号同步
  • 广告追踪

你的选择

你可以随时在 iOS 系统设置中关闭位置、日历、提醒事项或通知权限。关闭后，对应功能可能无法正常工作。
