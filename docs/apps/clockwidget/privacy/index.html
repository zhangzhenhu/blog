我们会使用的权限

位置 日历 提醒事项 通知
  • 位置：用于当前位置天气与天气摘要；仅在你使用相关功能时请求。
  • 日历：用于读取和展示系统日程，以及通过系统界面创建新日程。
  • 提醒事项：用于读取、展示和管理系统提醒事项。
  • 通知：用于计时器提醒、阶段切换提醒等本地通知能力。

数据如何处理

主要在设备本地处理的数据

  • 日历事件
  • 提醒事项
  • 计时器配置与运行状态
  • 小组件共享缓存

当前版本没有自建账号系统，也没有把上述内容上传到开发者自建服务器。

可能发送到第三方服务的数据

天气功能可能会将查询所需的位置数据发送给天气数据源，以返回天气结果。当前项目已接入或预留以下天气源能力：

  • Open-Meteo
  • Apple WeatherKit

当前未接入的能力

  • 广告 SDK
  • 行为埋点 / 统计分析 SDK
  • 第三方登录
  • 跨设备账号同步
  • 广告追踪

如果后续版本接入上述能力，本页面会同步更新。

你的选择

你可以随时在 iOS 系统设置中关闭位置、日历、提醒事项或通知权限。关闭后，对应功能可能无法正常工作。
